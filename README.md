* System Requirments: Windows 10

* Files : 3 `.Java` File and 1 `Documentation` directory
* JDK Version: 15.0.2 

## Install

```cmd
$ cd 2019213232_WINDOWS 
$ javac javac CheckIn.java
$ java CheckIn
```
